Formularios en HTML


Se usara la etiqueta form y sera mostrada de la siguiente forma

form class"" action"nombre del proyecto" method="post" cerramos form
En medio de form vamos a colocar nuestra primer etiqueta label en la cual dentro de ella se va a colocar lo que queremos que el usuario proporcione
Seguido de la funcion input text que permitira al usuario la opción de escribir y despues podemos agregar otro input submit para que se guarde lo anotado

De igual forma tenemos muchas opciones de input como las siguientes:

Un uso de los forms puede ser un Contact me area, en la cual habrá un espacio para poner nombre, correo electronico y un cuadro de texto donde el usuario podrá poner su petición
En este caso se usaria la etiqueta

textarea name="" rows="" cols=""

Donde textarea da la entrada al cuadro de texto
rows determina la altura del cuadro
y cols determina lo ancho del mismo

De momento nuestro boton submit no tiene uso ya que dentro de la función action"" en la etiqueta form la orden es regresar al menú principal
Action determina donde será mandada la informacion
Sin embargo para que tenga un uso real se necesita de CSS o Java

Lo que podemos hacer mientras, es dar una funcion basica, como podria serlo mandar la info a un correo electrónico usando "mailto:" y de igual forma usar enctype="text/plain" que su función será validar que nuestro sitio web está codificado en UTF-8

A su vez hay que poner en cada input el nombre de lo que se está haciendo ya que este dará el valor o sentido al mismo
